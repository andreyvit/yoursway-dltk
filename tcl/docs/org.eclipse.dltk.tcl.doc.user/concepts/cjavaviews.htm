Tcl views

The Tcl development tools contribute the following views to the workbench:

Script Explorer View

The Script Explorer view shows a Tcl-specific view of resources inside Tcl projects in your workbench. The Tcl element hierarchy is defined by the project's build class paths and consists of elements like

  • source folders and libraries
  • packages
  • source code files
  • types, methods and fields

Tcl Documentation View

This view shows documentation for the currently selected Tcl element (the documentation may be provided either in the form of comments or in manual pages).

View Customization

Both the appearance and behavior of the Tcl views can be customized.

  • Filtering
  • Sorting
  • Presentation options
