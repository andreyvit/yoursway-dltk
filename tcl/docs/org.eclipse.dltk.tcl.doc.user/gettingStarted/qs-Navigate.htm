Navigate to a Tcl element's declaration

In this section you will learn how to easily find a "declaration" of a referenced Tcl variable (usually, a set command for the variable).

  1. Create a new project as shown in the first secion o this tutorial.

  2. Open http.tcl file from http1.0 package (it can be found in the Interpreter libraries section of the project in the Script view).

  3. In the editor go to the line 27 (you can go to a specified line by selecting Navigate > Go to line menu item.):
    ...
    foreach name $options {
    ...

  4. Set cursor to the $options variable reference.

  5. Select Navigate > Open declaration menu item.

    Now the declaration of the variable is selected in the editor.
    Note that you might also use F3 key instead of the menu item to achieve the same effect.
