Tcl development overview

The DLTK Tcl/Xotcl project provides the tool plug-ins which implement Tcl IDE supporting Tcl and XOTcl application development. It adds a Tcl project nature and Tcl perspective to the Eclipse Workbench as well as a number of views, editors, wizards, code assistance tools and a builder.

Try the Basic tutorial to get a first impression. Discover more in the Tips and Tricks section.
