Applying refactoring scripts

  1. Select Refactor > Apply Script... from the refactoring menu to show the apply refactoring script wizard.
  2. On the first page of the Apply Script wizard, specify the refactoring script that you would like to apply to your workspace. You can use the clipboard as input or click on Browse... to browse for the location where the refactoring script is stored.
  3. Once you have entered a valid refactoring script, click Next to continue.
  4. On the second page of the Apply Script wizard, the wizard displays a preview of the refactorings to be replayed. Drill down to an individual refactoring and select it to see more details about the performed refactoring in the details view below.
  5. Click OK to replay the refactorings and close the dialog.
  6. Click Next to replay the refactorings and preview each refactoring before applying it to the workspace.
